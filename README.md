# animeko-guide
由于很多小白用animeko碰到各种问题，无法自己解决，于是便有了这个仓库。
# animeko是什么？
animeko是一个基于bangumi信息平台的在线追番的免费开源软件。
- 基本功能：追番进度同步/在线看番/番剧下载/聚合弹幕
- 运作流程：从bangumi获取条目信息 >从多个动漫网站搜索条目 >解析获取的剧集列表 >解析视频链接并播放
> 所以别再问 怎么切换分辨率？怎么切换字幕？之类的问题了，视频来自第三方网站，画质与字幕取决于来源网站。
